컬러 테마

설정 내보내기 / 가져오기

가져오기를 하면 현재 브라우저에 저장된 설정이 가져온 파일의 내용으로 덮어씌워집니다.

공연 좌석·티켓·메모 JSON 내보내기 / 가져오기

각 공연은 시간순으로 s1, s2, s3... 같은 숨겨진 ID를 가지고 있습니다.
형식: {"sid": [좌석, 티켓종류, 수수료여부, 메모]} — 리스트 첫 항목이 좌석이고, 나머지는 티켓종류·수수료여부(true/false)·메모 순입니다. (임의 할인권은 5번째에 할인율을 더 붙입니다: [좌석, 이름, 수수료, 메모, 할인율])
티켓과 메모가 모두 없으면 ["1-7-23"]처럼 좌석 하나만 넣습니다. 예: {"s2": ["1-7-23", "청소년", true, "1열 명당석"], "s30": ["1-6-24"]}
가져오기를 하면 기존 좌석·티켓·메모를 전부 지우고 가져온 내용으로 다시 채웁니다.

또는 JSON 텍스트를 직접 붙여넣기 (모바일 엑셀에서 복사한 깨진 JSON도 인식합니다)
