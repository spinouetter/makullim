스케줄 보기 옵션

스케줄을 가로로 스크롤할 때 어떤 회차인지 쉽게 알 수 있도록 돕는 옵션입니다.

티켓 옵션

켜면 좌석 입력칸 옆에 + 버튼이 생겨 티켓을 추가·정렬할 수 있습니다. 맨 위 티켓이 스케줄에 표시됩니다.

통계·가격·좌석표에는 맨 위 티켓만 반영됩니다. 추가 티켓은 관리·기록용이며 총 금액이나 관극 횟수에 합산되지 않습니다.

배역

개발 중인 기능 보기

아직 다듬는 중인 기능들입니다. 필요할 때 켜서 써보세요.

켜면 상단에 Finale 탭이 생겨 배역별 배우·관극 좌석을 한 장의 이미지(SVG/PNG/JPG/PDF)로 저장할 수 있습니다.

아직 다듬는 중인 기능입니다. 켜면 티켓 팝오버에 시계 아이콘(예매일 입력·취소료 계산)이 생기고, 상단에 Tickets 탭이 나타나며, 티켓 취소 시 계산된 취소 수수료가 자동 입력됩니다.

좌석 보기

팝업에서 손가락(핀치)·드래그, 또는 + / − 버튼으로 확대·이동할 수 있습니다. 선택한 좌석이 가운데에 오도록 시작합니다.

데이터 내보내기 / 가져오기

화면 설정과 좌석·티켓·메모가 모두 한 파일에 저장됩니다. 가져오기를 하면 현재 브라우저의 내용이 가져온 파일로 업데이트(덮어쓰기)됩니다.

구 버전 방식의 JSON 내보내기 / 가져오기

엑셀 정산판과 같이 단순히 좌석을 저장하고 읽어오는 기능을 합니다. 최신 버전의 모든 기능을 저장하기 위해서는 위의 데이터 내보내기 / 가져오기를 이용해주세요.
가져오기를 하면 기존 좌석·티켓·메모를 전부 지우고 가져온 내용으로 다시 채웁니다.

또는 JSON 텍스트를 직접 붙여넣기 (모바일 엑셀에서 복사한 깨진 JSON도 인식합니다)

저장 초기화

설정 초기화: 화면 설정·필터·테마·통계 구성 등만 초기화하고, 입력한 좌석·티켓·메모는 유지합니다. (개발 중 기능으로 저장값이 꼬였을 때 사용)
좌석 데이터 삭제: 입력한 좌석·티켓·메모만 삭제합니다. (설정은 유지)
