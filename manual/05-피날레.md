# 5. Finale

> Settings → "개발 중인 기능 보기" → **"Finale 보기"** 를 켜야 상단 탭에 나타납니다(기본 OFF, **Tickets 탭의 스위치와는 별개**입니다).

막공(마지막 공연) 관극 인증용 이미지를 만드는 화면입니다. 실제 극장 로비의 캐스트보드(정산판)를 본뜬 디자인에 배역별 배우 이름·사진·관극 횟수를 자동으로 채우고, 좌석 영역에는 나의 관극 히트맵을 넣어줍니다. SNS 공유용으로 설계됐습니다.

## 사용 방법

1. 상단 "통계 기준"에서 최초/공연시작/공연 중 변경 반영/비중 반영 중 하나를 고릅니다(Statistics 탭과 옵션 이름은 같고, Finale 탭을 처음 열 때 **초기값은 Statistics에 저장된 값을 그대로 물려받지만**, 그 이후 어느 한쪽에서 바꿔도 서로 실시간으로 연동되지는 않는 별개 설정입니다. 여기서 바꾼 값은 새로고침하면 초기화됩니다).
2. 갤러리에서 디자인 카드를 클릭합니다.

   > ⚠️ 갤러리에 처음 보이는 이미지는 배포 시 미리 만들어 둔 **예시(무작위 데이터) 미리보기**일 수 있습니다. **실제 내 관극 기록이 반영된 진짜 이미지를 보려면 카드를 클릭해 크게 보기**를 열어야 합니다.

3. 크게 보기 화면에서 핀치/휠로 확대, 드래그로 이동할 수 있습니다.
4. 상단 버튼으로 **PNG / JPG / PDF**로 저장할 수 있습니다(한글은 PDF에도 글자 그대로 담깁니다). 닫기 버튼이나 배경을 눌러 닫습니다.

## 알아두면 좋은 점

- 사진이 없는 배우는 자동으로 플레이스홀더 이미지가 들어갑니다.
- 슬롯(자리) 수보다 배우가 많아 배치가 안 된 경우가 있을 수 있습니다(눈에 보이는 오류는 아니고 조용히 넘어갑니다).
- 숫자를 직접 클릭해 값을 고치는 기능은 없습니다 — 관극 기록 자체를 바꾸면 이미지에도 반영됩니다.
