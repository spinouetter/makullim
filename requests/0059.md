## 요청 (작성: Spinouetter)

<!-- 대화로 전달된 요청을 Claude가 옮겨 적음 -->

- 막올림을 makullim으로 바꾸어줘. domain 및 localStorage를 제외한 모든 부분에서.
- 한국어 이름은 막울림이야.

---

## 정리 (작성: Claude)

- 앱 이름을 **막울림(makullim)** 으로 변경. 다음은 **보존**(요청의 예외 + 불변 식별자):
  - **도메인** `makollim.com`(CNAME·문서의 도메인 표기)
  - **localStorage 키**: `makollim:state:v1:<id>`, `makollim:dbx:*`, `makollim:dropboxAppKey`
    (키를 바꾸면 기존 기록·Dropbox 연결이 끊기므로 키 문자열은 그대로, 문서의 키 표기도 유지)
  - **GitHub 저장소 URL** `github.com/spinouetter/makollim`(문의 링크 — 실제 저장소명)
  - **requests/·HISTORY.md의 과거 기록**(작업 이력은 다시 쓰지 않음)
- 변경된 것:
  - UI·탭 제목: "막울림", "막울림 · {공연 제목}" (index.html·app.js)
  - 전역 변수: `MAKOLLIM_SHOW_ID/BUILD/CONFIG` → `MAKULLIM_*` (app.js·dropbox.js·config.js·
    pages.yml 주입부·index.html 마커 주석·workflows README — 배포 시 스텁과 코드가 같은
    커밋으로 함께 바뀌어 정합)
  - `window.makollimDropbox` → `makullimDropbox`(외부 참조 없음 확인)
  - 다운로드 파일명: `makullim-finale-*`(finale.js·gen-finale-preview.js),
    `makullim-settings-seats-*`·`makullim-seats-*`(app.js),
    Dropbox 백업 파일명 폴백 `"makullim"`(실백업은 APP_ID 기준이라 영향 없음)
  - 문서·주석: README(제목·본문), CLAUDE.md(이름 표기), palette-preview.html, 코드 주석의 막올림

## 결정 사항 (작성: Claude)

- localStorage 키·도메인·저장소 URL은 **외부 상태와 묶인 식별자**라 이름과 분리해 유지.
  README·CLAUDE.md에서 키 문자열을 언급하는 부분은 실제 키(makollim:…)를 그대로 표기.
- Dropbox 토큰 키(`makollim:dbx:*`)도 localStorage이므로 유지 — 기존 연결 끊기지 않음.
