## 요청 (작성: Spinouetter)

내 막공 기능을 추가해줘. 막공 설정에서 고를 수 있는 전체 옵션이야. 이걸 켜면 원래 캐스트 막공이랑 티켓이 입력된 공연 중에서 막공을 볼 수 있어. 이 때 숨기기를 하면 내 막공과 전체 막공 다 보여
이 걸 켜면 현재 막공 컬러 스킴이 내 막공에 적용돼
그리고 전체 막공 (페어막 포함) 색이 지금 보다 연하게 표현되고 알약은 선이 아니라 조금 긴 점선이 될거야. 구현하고 스크린샷 보여워

---

## 정리 (작성: Claude)

- "막공 표시" 설정에 **전체를 아우르는 새 옵션 "내 막공"**을 추가. 켜면 배역별 막공을 **두 겹**으로 보여준다:
  - **내 막공**: 티켓이 입력된(`hasTopTicket`) 회차만 대상으로 계산한 배역-배우별 마지막 출연 — 기존 강조 색(굵은 배경 알약)을 그대로 사용.
  - **전체 막공**: 기존 방식(전체 스케줄 기준) 그대로 계산하되, 옅은 색 + **점선** 알약으로 낮춰서 함께 표시.
- 둘이 **같은 회차**로 겹치면 평소엔 내 막공만 보여주고(중복 방지), 단 그 회차의 **티켓이 숨김 상태**(`ticketHidden`)면 "확실하지 않다"는 의미로 **둘 다** 표시.
- 페어막·주역 페어막의 알약(오벌 테두리)도 "전체 막공"에 해당하므로, 내 막공 옵션이 켜지면 같은 방식(옅은 색+점선)으로 낮아진다(페어막 자체의 "내 버전" 계산은 만들지 않음 — 시각적 낮춤만).

## 결정 사항 (작성: Claude)
- "티켓이 입력된 공연 중에서 막공"의 판정은 **좌석 유무**(`hasTopTicket`)만 본다 — 추가 좌석(다중 티켓)은 대상에 포함하지 않음(맨 위 티켓 기준, 배역별 막공 자체가 맨 위 티켓의 캐스팅 셀 기준이므로 일관).
- 페어막·주역 페어막은 "내 버전"을 별도로 계산하지 않고 **시각적으로만 낮춤**(옅은 색+점선) — 페어 쌍을 티켓 기준으로 재계산하는 것은 요청 범위를 벗어난다고 판단.
- "숨기기를 하면 내 막공과 전체 막공 다 보여" 구절은 **티켓 숨김(`ticketHidden`) 상태**를 가리키는 것으로 해석했다: 내 막공과 전체 막공이 같은 회차로 겹치는데 그 회차 티켓이 숨김이면(참석이 불확실하므로) 안전하게 둘 다 표시. 이 해석이 다르면 알려주면 다시 맞출게.
- 점선 간격(“조금 긴 점선”)은 CSS `border-style:dashed`의 브라우저 기본 렌더링을 사용 — border 속성만으로는 대시 길이를 정밀 제어할 수 없어 SVG 없이는 "조금 더 긴" 간격을 정확히 맞추기 어려움. 필요하면 SVG 테두리로 바꿔 길이를 조정할 수 있다.
