## 정리 (작성: Claude)
- 가로 스크롤 플로팅 오버레이(float-label)에서: ① 다음 공연(current-perf)도 하이라이트 색 적용(선택은 이미 됨), ② 다음 공연·선택 둘 다 오버레이 왼쪽에 막대 표시(스크롤 전 date-cell과 동일).

## 할 일 (작성: Claude)
- [x] `tr.current-perf .float-label` 배경 골드 틴트 + 왼쪽 막대(inset 3px gold)
- [x] `tr.row-highlight .float-label` 왼쪽 막대 추가(inset 4px blue) — 배경색은 기존 유지
- [x] 헤드리스 확인(현재공연=골드 bg+막대, 선택=블루 bg+막대)

## 결정 사항 (작성: Claude)
- 왼쪽 막대는 스크롤 전 date-cell의 inset box-shadow와 동일 두께/색(골드 3px, 블루 4px)으로 float-label에 inset box-shadow로 재현. (0017의 오른쪽 그라데이션 ::after와 충돌 없음)
- 오버레이 배경은 panel과 블렌딩해 불투명 처리(아래 내용 비침 방지).
