- 마티네 일정 색을 녹색으로.

  구분하면 좋을까? 이 때 보는 사람도 있어서 흐리게 하는 건 별로고.

---

## 정리 (작성: Claude)

- **마티네 녹색** — 기존 날짜색 체계(일·공휴일 빨강 `#e0594a`, 토 파랑 `#5b8fd6`)에
  마티네 녹색(`#4fae72`)을 추가. 마티네 판정은 기존 `isMatinee`(시작 17시 이전 +
  주말·공휴일 아님) 그대로 사용하므로 빨강·파랑과 충돌하지 않는다(주말·공휴일 우선).
  적용 위치는 날짜색과 동일: 스케줄 날짜·시간 셀, 플로팅 날짜 라벨, 좌석맵 회차 목록.
- **전관 구분** — 흐림(음영) 없이 구분하는 시안 4종을 제시(A 배지 / B 자물쇠 아이콘 /
  C 행 틴트+왼쪽 스트라이프 / A+C 조합). 사용자 선택 대기.
  데이터는 `schedule.json` 회차에 `"buyout": true` 필드를 추가하는 데이터 주도 방식 예정.
