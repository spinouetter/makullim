## 요청 (작성: Spinouetter)

<!-- 대화로 전달된 요청을 Claude가 옮겨 적음 -->

- 스케줄 화면에 "막공" 설정을 추가. 누르면 시트맵 필터 같은 오버레이가 뜬다.
- 오버레이에서 두 옵션을 각각 또는 둘 다 켤 수 있다:
  - **배역별 막공** — 각 배우가 자신의 배역에서 마지막으로 공연하는 날 강조
    (모서리가 둥근 배경색 사각형으로 둘러짐).
  - **페어막** — 배역 통계 추가처럼 배역을 순서대로 선택. 순서는 옵션 on/off와 무관하게 유지.
    - 켜면 배역 컬럼 순서가 바뀌어 선택 배역이 맨 앞으로. 숨겨진 배역은 자동 활성화되고
      숨김 기능이 작동하지 않는다.
    - 선택 배역의 모든 배우(조합)가 마지막으로 함께 공연하는 날을 타원형 박스 하나로 묶는다.

## 정리 (작성: Claude)

- 스케줄 옵션 바(지금·변경 내역 옆)에 **막공** 버튼 추가 → `#lastShowModal` 오버레이
  (시트맵 필터와 같은 `seat-overlay` 방식, 바깥 클릭·닫기·확인으로 닫힘, 변경 즉시 반영·저장).
- **배역별 막공**: `buildLastShowMap()`이 배역-배우별 마지막 출연(비중>0) 회차를 계산,
  해당 셀의 이름을 골드 배경 둥근 사각형(`.name.last-show`)으로 강조.
- **페어막**: `buildPairLastSet()`이 선택 배역들의 '그날 배우 구성' 조합별 마지막 회차를 계산,
  그 행의 선택 배역 셀들을 타원(알약) 테두리 하나로 묶음(`.pairlast(-start/-end)::after`).
  선택 배역은 표 맨 앞으로 재정렬(선택 순서), 배역 숨김은 무시(전 배역 표시 + 숨기기 버튼,
  숨김 바에서 배역 항목 제거). 배역 선택 UI는 배역 통계 추가와 같은 chip+순서 배지.
- 기준은 스케줄 **전체**(필터와 무관). 배역이 미정인 회차는 페어 조합에서 제외(미래 회차 등).
- 설정 3종(`lastShowRoleOn`·`lastShowPairOn`·`lastShowPairRoles`)은 localStorage에 저장·복원.

## 결정 사항 (작성: Claude)

- **막공 판정은 필터·표시 상태와 무관하게 전체 스케줄 기준** — 필터로 행을 걸러도
  "마지막 회차"의 의미가 바뀌지 않도록.
- **페어 조합 키는 배역별 배우 목록(이름 정렬 병합)** — 한 배역에 배우가 여럿(변경 반영)인
  날도 구성이 같으면 같은 조합으로 본다. 배역이 미정(배우 없음)인 날은 조합 미성립으로 제외
  — 미래 미정 회차가 전부 "마지막"으로 잡히는 것을 방지.
- **타원은 셀 3분할 테두리로 구현** — 선택 배역 셀들이 항상 연속(맨 앞 재정렬)임을 이용해
  시작/중간/끝 셀이 위·아래/왼쪽/오른쪽 테두리를 나눠 그려 하나의 알약처럼 보이게.
- **페어막 배역 선택은 chip 클릭 순서**(배역 통계 추가와 동일 UX), 토글 off로는 지워지지
  않고 오버레이의 초기화 버튼으로만 리셋.
- 숨김 상태(`scheduleHiddenCols`)는 건드리지 않음 — 페어막을 끄면 이전 숨김이 그대로 복원.
