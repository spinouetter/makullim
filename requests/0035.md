## 요청 (작성: Spinouetter)

(대화로 진행한 연속 요청을 정리)

- 다운로드 정비: SVG 저장 제거, PNG/JPG가 배경만 나오는 문제 수정, PDF는 한글을 **벡터**로. 사진은 위쪽 크롭인데 PDF에선 찌그러지니 맞추기.
- Finale에서 쓰는 글꼴이 사용자 환경에 의존하는지 점검하고 자급자족으로.
- Finale를 **여러 디자인 썸네일 갤러리**로: 클릭하면 크게 보기 오버레이(상단 고정 저장 버튼·닫기·이동 제한), placeholder 여러 개, refresh마다 랜덤 순서. 테두리 없이 단색 사각형, 사진 사이 약간 여백, 모바일은 열 제한, placeholder 색 다양·비율 랜덤(면적 비슷).
- 피날레 탭 설명 제거, 옵션 바 고정. 오버레이는 최소 배율 고정·여백 밖 스크롤 금지, 스마트폰 뒤로가기로 닫기.
- 사진 없는 배우 플레이스홀더는 세로 중앙 정렬.
- 미리보기 썸네일을 **GitHub Action으로 생성**(랜덤 데이터 모드). 시드는 마지막 공연 id, 시드 없으면 랜덤·텍스트 시드 허용. 실제 관극 데이터가 있으면 경고. 탭에선 동적 생성 안 하고 썸네일 없을 때만 폴백, 실제 렌더는 오버레이 열 때.

## 정리 (작성: Claude) — 코어(js 등) 변경 요약

> 정산판 이미지(`images/finale-board.svg`) 자체 편집(검은 프레임 제거·viewBox 크롭·"커버" 텍스트 삭제·사진 정렬 속성 등)은 이 문서에 남기지 않음. 아래는 **코어 로직(js/css/워크플로)** 변경.

- **다운로드/내보내기 개편** (`finale.js`)
  - SVG 저장 버튼·로직 제거(PNG 기본).
  - PNG/JPG: 외부 `<image href>` 사진을 **base64 data URI로 임베드**(캔버스에서 상대경로 사진이 안 나와 배경만 나오던 문제 해결).
  - PDF: 한글을 **벡터 텍스트**로. CSS 클래스 색·획을 계산값으로 속성에 "굽고" @font-face 제거, jsPDF에 **IBM Plex Sans KR·Anton TTF 임베드**. 사진은 슬롯 비율로 미리 크롭해 svg2pdf 왜곡 방지. 실패 시 고해상도 이미지 PDF 폴백.
  - **글꼴 자급자족** — 배우 이름·날짜/공연장이 원본에서 IBM Plex Sans KR(미임베드)라 사용자 OS 글꼴로 대체되던 것을 **로컬 woff2/ttf 임베드**로 통일(미리보기·PNG·JPG·PDF 동일). PDF도 CDN(NanumGothic) 대신 로컬 TTF.
- **디자인 썸네일 갤러리** (`finale.js`·`styles.css`) — 라이브 보드 직접 표시 → **저스티파이드 썸네일 갤러리**(실제 캐스트보드 1 + placeholder 5, refresh마다 랜덤 순서). placeholder는 테두리 없는 단색 사각형(색 다양·비율 0.75~1.35 랜덤), 같은 높이 줄로 여백 없이 채움(모바일 한 줄 2장). 클릭 시 크게 보기 오버레이.
- **크게 보기 오버레이** — 상단 고정 바(PNG/JPG/PDF·닫기, 스크롤 안 됨) + 확대/이동 뷰포트. **최소(맞춤) 배율에선 고정, 여백 밖으로 오버스크롤 금지**. **스마트폰 뒤로가기로 닫기**(history pushState/popstate).
- **피날레 탭 정리** — 설명 문구 제거, **통계 기준 옵션 바 고정**(썸네일만 스크롤). 플레이스홀더 **세로 중앙 정렬**(실제 사진은 상단 정렬 유지).
- **지연 렌더** — 탭에선 보드를 동적 렌더하지 않음. **오버레이 열 때(또는 썸네일 이미지가 없어 폴백할 때)만** 라이브 보드를 1회 렌더(캐시).
- **랜덤 데이터 모드 + 썸네일 자동 생성**
  - `?randomData` / `?randomData=<시드>` — 시드 없으면 랜덤, **텍스트 시드 허용**, 같은 시드=같은 결과(mulberry32). **실제 관극(좌석) 데이터가 있으면 적용 안 하고 경고.**
  - 끝난 공연의 약 40%에 **무작위 좌석을 배정**한 뒤 통계·좌석 히트맵을 정상 계산 → 배역 관극수·총합·좌석이 서로 **일관**.
  - `scripts/gen-finale-preview.js` — 앱을 띄워 보드를 렌더해 `images/finale-preview.png` 생성. **시드 = 마지막(가장 최근 끝난) 공연 id** → 사소한 커밋엔 동일 이미지, 공연이 끝나면 갱신.
  - `.github/workflows/pages.yml` — Pages 배포(=커밋 push)마다 + 매일 cron으로 미리보기 재생성해 배포물에 포함(커밋 안 함, `.gitignore`). 탭 썸네일이 이 이미지를 쓰고 **없으면 라이브 보드로 폴백**.

## 결정 사항 (작성: Claude)
- **여백 없는 갤러리 = 저스티파이드 줄**(같은 높이). 면적 완전 균등(±1.2배)은 비율 유지 시 틈이 생겨 포기, 대신 비율 범위(0.75~1.35)를 좁혀 크기 차 최소화.
- **미리보기는 Pages 빌드 시 생성·비커밋** — 랜덤 바이너리를 커밋하지 않고 배포물에만 포함(로컬/PR에는 없어 폴백). 시드를 마지막 공연 id로 고정해 결정적으로.
- **랜덤 데이터 모드는 관극 데이터 없을 때만** — 실데이터 위에 가짜를 덮지 않도록 경고 후 무시. 좌석을 무작위 배정하고 통계는 정상 경로로 계산해 수치 일관성 확보.
- **PDF 한글 벡터** — svg2pdf가 @font-face를 못 읽어 CSS를 속성으로 굽고 TTF를 직접 임베드. 사진은 미리 크롭(svg2pdf가 slice 무시).
