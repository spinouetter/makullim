## 요청 (작성: Spinouetter)

- 좌석 미리 보기 개선
  - 설정 창에 기본 배울 설정 기능 추가 10~100%로 10% 단위
  - 핀치 투 줌, 드래그 지웜
  - 데스크탑 환경을 위해 맵 위에 + - 버튼 추가

## 정리 (작성: Claude)
- 대상: **좌석 보기 팝업(`#seatOverlay`)만**. 메인 '좌석맵' 탭은 이미 줌·핀치·드래그·+−가 있어 그대로 둠.
- 현재 팝업은 고정 viewBox(맵 전체)·정적 SVG라 확대/이동 불가 → 팝업에 **핀치 줌 + 드래그(팬) + 데스크탑 +/− 버튼** 추가.
- **설정에 "좌석 보기 기본 배율"** 추가: 1배(전체 맵)~5배. 팝업 열 때 이 배율로 시작.
- 팝업 열 때 **선택(맨 위) 좌석을 가운데에** 두되, 구석 좌석은 맵 경계에 맞춰 클램프(완전 중앙 아니어도 됨).

## 할 일 (작성: Claude)
- [x] 설정에 "좌석 보기 기본 배율" 선택(1~5배) 추가 + 저장/복원(`previewDefaultZoom`)
- [x] `buildSeatSvgInner`가 맨 위 좌석의 렌더 위치(`topPos`) 반환 → 팝업 중심 정렬에 사용
- [x] 팝업 SVG에 줌/팬 상태(viewBox) + 초기 배율(설정값) 적용, 컨테이너 비율에 맞춤
- [x] 팝업 상호작용: 드래그 팬 · 핀치 줌 · 휠 줌 · +/− · 전체(1배) 버튼
- [x] 줌 범위 클램프(1배=전체 ~ 최대 8배), 맵 밖으로 안 나가게 위치 클램프
- [x] 헤드리스 확인(초기 배율·팬·줌·버튼, JS 에러 없음)
- [x] (보완) 좌석이 여러 개(다중 티켓)면 기본 배율 대신 **모든 좌석이 다 보이게** 묶음 맞춤(여유 패딩 포함)

## 결정 사항 (작성: Claude)
- 배율 의미: **전체 맵=1배, 확대 2~5배**(사용자 확정). 설정 기본값 2배(전체 보고 싶으면 1배 선택). 수동 줌은 약간 여유(최대 ~8배)까지 허용.
- 팝업 SVG는 메인 좌석맵과 별개 상태로 자체 줌/팬 구현(전역 mainViewBox와 분리). `preserveAspectRatio`는 기본(meet), viewBox 비율을 팝업 컨테이너 비율에 맞춰 계산.
- 중심 정렬: 선택 좌석 중심 + 맵 경계 클램프(구석이면 경계까지만). 팝업은 항상 모든 층 표시(메인맵의 숨김층 설정 영향은 기존 동작 유지).
