## 정리 (작성: Claude)

- 통계 탭에 **"취소·변경 통계"** 섹션 신설(대결 통계와 기타 통계 사이, 접기 지원).
- **데이터 주도 표시 여부**: 공연 정의 `stats.disruptions: true`인 공연만 섹션 표시
  (빌리 2021만 켬). 코드에 공연 하드코딩 없음 — 새 공연도 플래그만 켜면 동작.
- 모든 수치는 스케줄 데이터에서 계산:
  - 취소 = `cancelled` 플래그, 일정 변경 = `reschedules` 이력
  - 캐스팅 변경 = 비중 0 항목(`[[공지,0],실제]`, 공지됐다 빠진 배우)
  - 공연 중 변경 = 한 배역에 비중>0 배우 2명 이상(예: 데비 이윤슬5:성아인5)
- 표시 블록 4종:
  1. **요약** — 전체/취소(비율)/일정 변경(이력 건수)/캐스팅 변경/공연 중 변경/
     **예정대로 진행된 회차**(아무 변동 없던 회차, "얼마나 힘들었는지"의 요약)
  2. **중단 구간** — 연속 취소 블록의 기간·일수·회차수(팬데믹 중단 2회 + 단발 중단)
  3. **배역·배우별** — 배우별 취소/변경(빠짐)/변경(투입)/공연 중 변경 횟수(배역 rowspan).
     공연 중 변경은 방향 기호로 구분: **▼ 공연 중 빠짐 / ▲ 중도 투입**(표 아래 범례)
  4. **공연 중 변경 목록** — 회차·배역·배우 흐름(예: 이윤슬▼ → 성아인▲)

## 결정 사항 (작성: Claude)

- **"변경"을 두 방향으로 분리**: 변경(빠짐)=공지됐다 못 오른 횟수 / 변경(투입)=남 대신
  투입된 횟수 — 한 셀의 취소선·실제 배우가 각각 한 쪽씩 갖는다. 배우 입장에서 의미가 다름.
- **취소 횟수는 취소 회차의 공지 캐스팅 기준**(그 배우가 서기로 돼 있던 취소 공연 수).
- **싱글 캐스트 배역의 취소 제외**: 정규 cast가 1명뿐인 배역(데드맘·브웨·토니·조지)은
  전 회차 출연이 기본이라 그 본인의 취소는 집계하지 않는다. **다만 그 배역 커버의 취소는
  표시**(예: 토니 커버 고철순의 취소 1회) — casts.json의 cast/cover 표기로 자동 판별.
- **"예정대로 진행된 회차"** = 취소·일정 변경·캐스팅 변경·공연 중 변경이 전혀 없던 회차 —
  시즌 전체의 험난함을 한 줄로 보여주는 핵심 지표(빌리 2021은 47.7%).
- 블록은 드래그 정렬 없이 고정(기타 통계와 달리 항목이 적어 불필요), 섹션 접기는
  기존 `collapse-h2[data-sec]` 메커니즘을 그대로 사용(상태 저장 자동).
- CSS 무변경 — 기존 `role-stat-table` 스타일 재사용.
- **모바일 폭 대응**: 배역·배우별 표 헤더를 축약(변경(빠짐)→빠짐, 변경(투입)→투입,
  공연 중 변경→공연 중)하고 배역·배우 셀 줄바꿈 방지 — 390px에서 이름 한 줄 표시,
  축약 의미는 표 아래 범례로 설명.
