## 정리 (작성: Claude)
- 통계 탭에 "기타 통계" 섹션 추가(접기 가능 = 기타 통계 닫기 버튼). 내부에 세부 블록 3종, 각 블록 접기(세부 통계 닫기)·드래그 이동(세부 통계 이동) — 대결 통계 블록 패턴 재사용.
- 월별 / 요일별 / 티켓별 통계.

## 결정 사항 (작성: Claude)
- 마티네 = 시작 17시 이전. 휴일 = `holidays.json`에 있고 주말(토·일) 아닌 날.
- 티켓별 컬럼: 관극(종료&좌석) / 예매(미종료&좌석) / 총액(해당 등급·티켓의 티켓금액 합). 좌석 등급은 `gradeOf(seat)`, 같은 등급은 rowspan으로 묶음. ("총액"은 금액 합으로 해석)
- 월별/요일별 컬럼은 앱 표준(전체/종료/관극/예매)과 통일.
- 세부 블록 접기·이동은 대결 통계(match) 블록 구현을 그대로 따름.
