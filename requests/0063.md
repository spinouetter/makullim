# 요청 #0063 — 스케줄 표 공연 순번 컬럼(연번·자번)

상태: 완료 <!-- 대기 / 진행 중 / 완료 / 보류 -->

---

## 요청 (작성: Spinouetter)

<!-- 대화로 전달된 요청을 Claude가 옮겨 적음 -->

- 시간 옆에 공연 번호 추가 해줘. 연번과 자번.
  연번은 취소 공연을 제외한 공연 번호, 자번은 좌석 등록한 번호.
- 각각 10의 배수, 50의, 100의 배수는 더 눈에 띄게.
- (후속) 숫자 레이블은 횟수로 변경. 숫자를 가운데 정렬해서 보여줘봐.

---

## 정리 (작성: Claude)

- 시간 컬럼 오른쪽에 **순번** 컬럼을 추가한다(숫자는 가운데 정렬).
  - **연번**: 취소 공연을 제외한 회차 순번(옅은 색). 취소 회차는 번호 없음.
  - **자번**: 좌석을 등록한(취소 제외) 회차의 순번(골드, `·` 구분점). 좌석 없으면 표시 안 함.
- 번호는 **필터와 무관하게 전체 스케줄 기준**으로 매기므로, 배역 필터를 걸어도 변하지 않는다.
- **단계 강조**: 10의 배수 = 굵게(연번은 밝은 색), 50의 배수 = 굵게+테두리 알약,
  100의 배수 = 굵게+골드 배경 알약(강조 알약엔 자번 구분점 생략).
- 다른 특수 컬럼(좌석·티켓·가격·메모)과 동일하게 헤더 "순번 ▾ → 숨기기"로 숨기고
  숨김 바에서 복원할 수 있다(`scheduleHiddenCols`의 `__num__`).

## 결정 사항 (작성: Claude)

- **자번도 취소 회차는 제외** — 취소된 공연에 좌석이 남아 있어도 관극 순번으로 세지
  않는다(연번과 기준 통일).
- 두 번호를 한 컬럼에 표시(`연번 ·자번`) — 컬럼 폭을 아끼고, 색(옅음/골드)으로 구분.
- 강조는 연번·자번 각각 독립적으로 적용(예: 연번 50 회차에 자번 23이면 연번만 알약).
