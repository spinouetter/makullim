## 요청 (작성: Spinouetter)

통계의 상위 메뉴에도 숨기기 기능 추가
- 배역 통계 추가와 같게

배역 통계 추가 UI 변경
- 배역 버튼의 크기 줄임. 텍스트 크기는 그대로 유지. 패딩 줄임

## 정리 (작성: Claude)
- 통계 상위 3개 섹션(배역별 출연 횟수 / 배역 조합 출연 횟수 / 대결 통계)에 "배역 통계 추가"와 동일한 접기(▾▸) 기능 추가. 접힘 상태는 저장됨.
- "배역 통계 추가"의 배역 선택 버튼(`.combo-chip`) 패딩 축소(`6px 14px`→`2px 10px`), gap `8px`→`6px`. 글자 크기(13px)는 유지.
