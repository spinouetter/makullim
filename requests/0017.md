# 요청 #0017 — 플로팅 가로스크롤 오버레이 플랫화

상태: 완료 <!-- 대기 / 진행 중 / 완료 / 보류 -->

---

## 요청 (작성: Spinouetter)

플랫한 가로스크롤 오버레이

현재 디자인은 플랫하나, 가로스크롤 오버레이를 켰을 때 생기는 오버레이는 플랫하지 않음
오른쪽의 그림자는 나쁘지 않으나 위아래 그림자 표시는 과한 것 같음

---

## 정리 (작성: Claude)
- 가로 스크롤 시 뜨는 플로팅 날짜 오버레이(`.float-label`)의 그림자가 `3px 0 8px`(blur 8)이라 위·아래로 번져 과함.
- 오른쪽 그림자는 유지하되 위·아래 번짐 제거 → 플랫하게.

## 할 일 (작성: Claude)
- [x] (1차) box-shadow 음수 spread → 오른쪽만 남김. 단 모서리 blur로 위·아래가 옅어지는 문제 남음.
- [x] (2차/최종) box-shadow 제거 후 `.float-label::after`에 세로 전체(top:0~bottom:0) 그라데이션 → **높이 전체 균일** 그림자
- [x] 헤드리스 확인(플로팅 옵션 켜고 가로 스크롤 → 오버레이 플랫·균일)

## 결정 사항 (작성: Claude)
- box-shadow는 blur 특성상 모서리에서 흐려져 위·아래가 옅어짐 → `::after` 그라데이션(`linear-gradient(to right, rgba(0,0,0,.30), transparent)`, width 8px)으로 대체해 세로 전체 동일 강도. border-right 1px는 유지(경계 구분).
