# 요청 #0049 — 막 내린 공연의 통계·스케줄 간소화

상태: 완료 <!-- 대기 / 진행 중 / 완료 / 보류 -->

---

## 요청 (작성: Spinouetter)

<!-- 대화로 전달된 요청을 Claude가 옮겨 적음 -->

- 종료된 공연은 통계에서 종료·남음·예매를 표시하지 않는다.
- 쓴 돈(금액 카드)도 총액만 표시.
- 스케줄에서 '지금' 버튼도 없애고, 지나간 공연 음영도 없앤다.

---

## 정리 (작성: Claude)

막을 내린 공연(예: 빌리 2021 아카이브)에서는 종료(=전체)·남음(=0)·예매(=0)가 전부
자명한 값이라 소음이므로, 공연 종료 판정 헬퍼 `isShowFinished()`를 두고 다음을 간소화:

- **통계 탭** — 상단 카드를 `전체·관극·총액` **한 줄(3칸)**로 합침(금액 카드 줄은 숨김),
  배역별·배역 페어·기타(월별·요일별) 표는 `전체·관극` 열만, 티켓별 표는 `예매` 열 제거.
- **스케줄 탭** — '지금' 버튼 숨김(현재 회차가 없으므로 무의미), 지난 공연 흐림(`past`
  음영) 미적용 — 아카이브에선 전 행이 지난 공연이라 표 전체가 흐려지는 문제 해소.

## 결정 사항 (작성: Claude)

- **종료 판정 = 모든 회차 종료 + `endDate` 경과** — 회차만 다 지난 경우(예: 엘리자벳이
  9/4까지만 등록된 상태로 9/4가 지난 시점)는 아직 회차가 추가될 수 있으므로 진행 중으로
  본다. endDate가 없으면 회차 기준만 사용.
- 티켓별 표의 `예매` 열도 제거 대상에 포함(종료 공연에선 항상 0), `총액`은 유지.
- 취소(`perf-cancelled`)·일정변경 표시는 아카이브에서도 사실 기록이므로 유지.
