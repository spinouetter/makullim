# 요청 #0010 — 스케줄 필터 고도화

상태: 완료 <!-- 대기 / 진행 중 / 완료 / 보류 -->

---

## 요청 (작성: Spinouetter)

스케쥴 화면 필터 고도화
  - 스케줄 화면에서 결과 필터를 적용한 경우 배역 오른쪽 위에 선택한 배우 갯수가 오버레이
  - 옵션에 필터 끄기 버튼 활성화 됨
    - 누르면 스케줄에 적용된 전체 필터 초기화

---

## 정리 (작성: Claude)

1. **필터 적용 배지**: 필터가 걸린 컬럼 헤더(배역, 그리고 대결 '결과 필터') 우상단에 **선택 개수** 배지 오버레이. (좌석맵 필터 배지와 유사한 스타일)
2. **필터 끄기 버튼**: 스케줄 툴바에 추가. 필터가 하나라도 걸려 있으면 **활성**, 없으면 비활성. 누르면 스케줄의 **모든 필터(배역 필터 + 대결 결과 필터) 초기화**. (숨긴 컬럼은 필터가 아니므로 유지)

## 할 일 (작성: Claude)
- [x] 배역/대결 헤더에 선택 개수 배지(`.col-filter-badge`, 우상단 골드) (`?v=31`)
- [x] 툴바에 `#filterClearBtn` "필터 끄기" (필터 유무로 활성/비활성, 클릭 시 `scheduleRoleFilter`+`scheduleMatchFilter` 초기화)
- [x] 헤드리스 검증: 필터 적용→배지(2)·버튼 활성, 대결필터→배지2개, 끄기→배지0·버튼 비활성. 스크린샷 확인.
- [x] 미세조정(Spinouetter, `?v=32`): 배지 크기 10%↓(15→13.5px, font 9→8), 위치를 이름과 살짝 겹치게 왼쪽으로(right -9→8px).
- [x] 재조정(Spinouetter, `?v=33`): 고정 right는 이름 길이에 따라 위치가 달라져 긴 이름(마이클/Mrs. 윌킨슨)을 가림 → **삼각형(▾) 기준**으로 배치(배지를 `.col-arrow`의 자식으로). 이름 길이와 무관하게 항상 이름 끝. 배지 **투명도** 추가(opacity 0.8). 헤드리스로 짧은/긴 이름 모두 확인.

## 결정 사항 (작성: Claude)
- 배지는 배역 필터(선택 배우 수)와 대결 결과 필터(선택 결과 수) 모두에 표시(일관성).
- "필터" = `scheduleRoleFilter` + `scheduleMatchFilter`. 숨긴 컬럼/하이라이트 등은 초기화 대상 아님.
- 버튼은 항상 보이되 필터 없으면 disabled(하이라이트 해제 버튼과 동일 패턴).
