## 정리 (작성: Claude)

Finale 탭은 그동안 Settings "개발 중인 기능 보기"의 **Finale 보기**(`optFinaleView`) 토글로만 노출됐다(기본 OFF). 이제 정식 기능으로 승격 → **항상 표시**하고 토글을 제거한다.

## 할 일 (작성: Claude)

- [x] `optFinaleView` 체크박스 와이어링(app.js) 제거
- [x] `index.html` — "개발 중인 기능 보기" 섹션의 **Finale 보기** 체크박스+설명 문단 삭제(Tickets 토글 `optDevFeat`는 유지)
- [x] `index.html` app.js 캐시 v206 → v207
- [x] 설정 없이도 Finale 탭 노출 + 클릭 시 갤러리 렌더 확인

## 결정 사항 (작성: Claude)
- `finaleViewOn` 변수·저장/복원(state)은 **그대로 둠**(무해). 과거 저장 상태에 `finaleViewOn:false`가 있어도 `applyFinaleVisibility`가 무조건 표시하므로 탭이 숨지 않음.
- "개발 중인 기능 보기" 섹션 자체는 유지(Tickets 탭 토글이 남아 있음).
